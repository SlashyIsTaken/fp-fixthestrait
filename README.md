## Overview

The Strait of Hormuz is a 33-kilometer-wide body of water between Iran and Oman through which approximately 21 million barrels of crude oil transit daily — roughly one-fifth of global oil consumption. It is, by every available metric, the most geopolitically anxious chokepoint on Earth.

This project is a single-page satirical dossier that proposes twelve increasingly impractical solutions to the chokepoint problem, each presented with the unwavering gravity of an actual classified briefing. It is also a live market intelligence terminal, an interactive group chat between nine sovereign states and one dolphin, a tanker simulator, and a small monument to the principle that, in times of genuine global instability, sometimes the most appropriate response is to build a website about training cetaceans to carry petroleum.

The site is fully static. There is no build step, no framework, no telemetry, no analytics beyond a single Google tag, and no opinions about energy policy that would survive five minutes of expert review.

## Features

### Intelligence Products
- **Crude Oil Situation Room** — live WTI crude pricing, intraday price and volume charts, secondary asset strip (Brent, Natural Gas, Gold, VIX), and the proprietary **Strait Incident Probability™** gauge, which is always elevated and always wrong
- **Ceasefire detection** — when WTI drops meaningfully and cross-asset signals (VIX, Brent, Gold) confirm genuine risk-off behaviour, the Situation Room enters a de-escalation posture: the probability gauge applies a stability discount, the analyst note switches to a dedicated grudging-disappointment pool, and a green CEASEFIRE banner appears in the price panel. Analysts remain unhappy about it.
- **Sources & Methods** — a dossier sub-section listing the intelligence sources from which all assessments are derived. Includes ████████, the vibes, an intercepted text message that just said "lol", and Steve. All sources rated MEDIUM-HIGH; the rating scale is one rung tall by design.
- **Classified Mode** — toggle to reveal hidden analyst notes, internal memos, intercepted communications, and a market intelligence assessment regarding Steve (dolphin)
- **Satirical news ticker** — rotating intelligence intercepts of varying credibility (range: low to extremely low)

### Interactive Briefings
- **Twelve solution dossiers** — each with classified analyst notes, real-world facts, and an interactive component
- **Email Composer** *(Solution 01)* — draft your own diplomatic correspondence to Iran, or review the one the intern already sent
- **Carrier Incident Log** *(Solution 04)* — observe the logistics of an aircraft-carrier bridge collapsing in real time
- **Canal Drawing Tool** *(Solution 08)* — sketch a bypass canal across the Arabian Peninsula with Leaflet
- **Iran Relocation Map** *(Solution 06)* — drag Iran somewhere more convenient
- **Strait Simulator** — pilot a tanker through obstacles including angry tweets, rogue dolphins, and floating NFTs
- **The Strait Room** — a fake group chat between the USA, Iran, the UAE, Oman, Saudi Arabia, Qatar, Bahrain, Kuwait, and Steve (dolphin)
- **Comparison Matrix** — all twelve solutions rated across cost, feasibility, dolphin involvement, and probable geopolitical fallout
- **What If? Timeline** — day-by-day projection of global economic collapse following hypothetical strait closure

### Reference Materials
- **Ask the Analyst (Steve)** — submit any question about the Strait, the oil market, or anything related, and receive an official non-answer from Steve's desk. Pure client-side, no LLM, no backend, no engagement; the bureaucratic non-engagement is the entire joke. Same question always returns the same case number and the same dead-eyed reply.
- **Glossary** — twenty deadpan definitions of terminology used elsewhere on the dossier (CHOKEPOINT, STEVE, SAFE HAVEN ASSET, OPEC+, *etc.*). Each definition sounds like it belongs in an actual textbook and contains exactly one off note.

### Quality of Life
- **Solution submission form** — submit your own solution; it will be rejected in increasingly creative ways
- **Hash-based deep linking** — share a direct link to any solution (`fixthestrait.com/#dolphins`)
- **Custom 404 page** — the missing page was last seen transiting the Strait of Hormuz

## Architecture

The project is intentionally austere. There is no bundler, no framework, no transpiler, and no `node_modules` directory. The entire site is hand-written HTML, CSS, and ES5-flavoured vanilla JavaScript.

| Layer | Technology | Notes |
|-------|------------|-------|
| Markup | HTML5 | Single-page, semantic, hand-written |
| Styling | CSS3 with custom properties | Dark monochrome dossier aesthetic, mobile breakpoint at ~700px |
| Scripts | Vanilla JS (ES5-friendly) | `var`, function expressions, no arrow functions, no build step |
| Maps | [Leaflet.js](https://leafletjs.com/) | Canal drawing, Iran relocation |
| Charting | Native Canvas API | Intraday price chart, volume bars, simulator |
| Live data | Cloudflare Worker | Proxies Yahoo Finance with 15-minute edge cache |
| Hosting | Docker + nginx | Reverse-proxied static delivery |

## Getting Started

### Prerequisites

- Docker, **or** any tool capable of opening an HTML file
- A working belief in the absurdity of geopolitics (recommended, not required)

### Run with Docker

```bash
docker compose up
```

The site will be available at [http://localhost:8080](http://localhost:8080).

### Run Without Docker

```bash
# macOS
open website/index.html

# Linux
xdg-open website/index.html

# Windows
start website/index.html
```

The site is fully static and will run from `file://` with one caveat: the Crude Oil Situation Room expects to fetch live data from a Cloudflare Worker. Without the Worker (or with CORS preventing the call), the Situation Room gracefully degrades to a satirical error message about the CIA intercepting the data feed, which is, honestly, fine.

## The Oil Data Pipeline

Live market data is served from a Cloudflare Worker deployed at `oil.fixthestrait.com`. A reference copy of the Worker source is checked into this repository as [`cloudflare-worker.js`](cloudflare-worker.js).

> **Important:** that file is a *reference copy only*. The live Worker is deployed via the Cloudflare dashboard or `wrangler deploy`. Editing the file in this repository does not redeploy anything. The user must copy the file into Cloudflare manually after changes.

The Worker proxies the Yahoo Finance chart endpoint, shapes the response into a stable schema, and caches the result for fifteen minutes at the edge. CORS is locked to `https://fixthestrait.com`.

### Tracked Assets

| Symbol | Asset | Role |
|--------|-------|------|
| `CL=F` | WTI Crude Oil | Primary — full intraday candle stream |
| `BZ=F` | Brent Crude Oil | Secondary — price and change only |
| `NG=F` | Natural Gas | Secondary |
| `GC=F` | Gold | Secondary (the actual fear gauge) |
| `^VIX` | CBOE Volatility Index | Secondary (the *other* actual fear gauge) |

To add another asset, append it to the `OTHERS` array in [`cloudflare-worker.js`](cloudflare-worker.js), add a snarky entry to `assetQuips` in [`website/oil.js`](website/oil.js), and redeploy the Worker.

## Contributing

Pull requests are welcome. Issues are welcome. Opinions on the structural feasibility of the trebuchet solution are especially welcome.

The world is, presently, a lot. There are real geopolitical crises, real market disruptions, and real people making real decisions about the Strait of Hormuz that meaningfully affect the global economy. This project does not address any of that. What it does, instead, is point at the absurdity of the situation and laugh — because, occasionally, that is the only sensible response available.

### Things that make good contributions

- A thirteenth (or fourteenth, or fifteenth) solution — the stupider the better, the more grounded in real facts the funnier
- Additional **Strait Room** chat batches; Steve in particular has more to say
- Additional news-ticker headlines
- Mobile UX improvements
- Translations — geopolitical anxiety is a universal language

### Editorial guidelines

- **Maintain the deadpan voice.** The site is funny because it never breaks character. Every piece of copy is delivered with the unwavering seriousness of a leaked intelligence memo. Do not soften jokes, do not wink at the camera, do not include the words "lol" or "haha" anywhere a user can see them.
- **Stay grounded.** The funniest jokes are the ones built on a real fact. The Strait is genuinely 33 km wide. Twenty-one million barrels really do transit it daily. The dolphin program is a real thing. Use real data wherever possible — it makes the absurdity hit harder.
- **No dependencies.** No npm, no bundler, no framework. ES5-friendly JavaScript only.
- **No unsolicited recommendations to actual policy makers.** The dolphins are not a serious proposal.
